# Flask Learning

### How to set up environment?

```
make install
```

### How to run project?

Run dev server
```
make run
```

Run prod server
```
make run-prod
```

### How to test project?

Run all tests, type checker, and linter
```
make test
```

Run unittests
```
make unittests`
```

Run coverage
```
make coverage
```

Run type checker
```
make types
```

Run linter
```
make lint
```

Run formatter
```
make black
```
